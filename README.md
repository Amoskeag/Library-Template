# Amoskeags's Cookie Cutter Library Template

I wanted to make library creation more automated for starting a new project.


## Using this template.

Its as easy as

```shell
cookiecutter https://github.com/Amoskeag/Library-Template
```

If you dont have cookie-cutter, install it from [here](https://cookiecutter.readthedocs.io/en/stable/installation.html)!


## Whats in this template?

```
.
└── YourProjectLibrary                      # This Library is best used for project specific parts. Things that are unique to your build.
    ├── 3D                                  # CAD Files, 3D objects of your electrical component.
    ├── Footprints                          # The footprints of your components placed on your PCB.
    └── Symbols                             # Symbols of your component, shown on the Schematic page when added.
    
```
